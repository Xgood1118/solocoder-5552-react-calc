## 1. 产品概述

网页版科学计算器，面向工程师和学生的纯前端工具。覆盖从基础四则到矩阵运算的完整科学计算需求，所有计算在浏览器本地执行，无需后端，支持 PWA 离线使用。解决系统计算器功能薄弱、Mathematica/MATLAB 过重之间的空白。

## 2. 核心功能

### 2.1 用户角色
| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 普通用户 | 无需注册 | 使用全部计算功能、管理历史记录与变量 |

### 2.2 功能模块
1. **主计算页面**：表达式输入与计算、科学计算键盘、结果展示
2. **复数运算页**：复数表达式计算、复数专用操作
3. **矩阵运算页**：矩阵可视化编辑、矩阵运算
4. **单位换算页**：10大类单位互转、进制互转
5. **历史与收藏页**：计算历史、公式收藏、用户变量管理

### 2.3 页面详情
| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 主计算页 | 表达式输入区 | 支持键盘输入和点击按钮输入，显示完整表达式，光标可移动 |
| 主计算页 | 科学键盘区 | 四则运算、括号、阶乘、组合排列、三角函数、对数、幂运算、常量等按键 |
| 主计算页 | 结果显示区 | 实时显示计算结果，错误时显示友好提示 |
| 主计算页 | 角度/弧度切换 | 切换三角函数的角度制与弧度制，默认弧度 |
| 复数运算页 | 复数输入区 | 支持 a+bi 格式输入，含 i 按键 |
| 复数运算页 | 复数操作区 | 模、辐角、共轭等专用操作按钮 |
| 矩阵运算页 | 矩阵编辑器 | 可视化行列调整，单元格直接填数字 |
| 矩阵运算页 | 矩阵操作区 | 加减乘、转置、求逆、行列式、特征值、LU/QR分解 |
| 单位换算页 | 类别选择 | 10大类单位切换标签 |
| 单位换算页 | 换算输入 | 左右双栏输入，实时互转 |
| 单位换算页 | 进制互转 | 2/8/10/16进制互转，支持小数 |
| 历史与收藏页 | 历史记录 | 倒序展示，点击回填，一键清空 |
| 历史与收藏页 | 公式收藏 | 预设常用公式，点击填入输入框 |
| 历史与收藏页 | 变量管理 | 用户自定义变量，本地持久化 |

## 3. 核心流程

用户打开计算器 → 选择计算模式（基础/复数/矩阵/单位换算） → 输入表达式或数据 → 点击计算或按回车 → 查看结果 → 结果自动存入历史记录 → 可将结果赋值给变量或收藏公式

```mermaid
flowchart TD
    "A[打开计算器]" --> "B{选择模式}"
    "B" --> "C[基础计算]"
    "B" --> "D[复数运算]"
    "B" --> "E[矩阵运算]"
    "B" --> "F[单位换算]"
    "C" --> "G[输入表达式]"
    "D" --> "H[输入复数表达式]"
    "E" --> "I[编辑矩阵数据]"
    "F" --> "J[选择单位类别]"
    "G" --> "K[计算并显示结果]"
    "H" --> "K"
    "I" --> "L[执行矩阵运算]"
    "J" --> "M[实时换算结果]"
    "K" --> "N[存入历史记录]"
    "L" --> "N"
    "N" --> "O{后续操作}"
    "O" --> "P[赋值给变量]"
    "O" --> "Q[点击历史回填]"
    "O" --> "R[收藏公式]"
```

## 4. 用户界面设计

### 4.1 设计风格
- 主色调：深灰黑底 + 琥珀色强调（模拟专业仪器质感）
- 次色调：白色/浅灰文字、暗色按键
- 按钮风格：圆角微凸（box-shadow 模拟 3D 按键手感），hover 时微沉
- 字体：等宽字体用于表达式（JetBrains Mono），UI 文字用 Noto Sans SC
- 布局：左为主计算区域，右为模式切换标签页，顶部为工具栏
- 图标：使用 lucide-react 图标库

### 4.2 页面设计概览
| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 主计算页 | 表达式输入区 | 等宽字体、大号显示、光标闪烁、底部分隔线 |
| 主计算页 | 科学键盘区 | 6行8列网格、功能键琥珀色高亮、数字键深灰 |
| 主计算页 | 结果显示区 | 右对齐大号字体、结果与表达式视觉区分 |
| 复数运算页 | 复数键盘 | 在基础键盘上增加 i/共轭/模/辐角按键 |
| 矩阵运算页 | 矩阵编辑器 | 网格表格样式、可拖拽调整行列、当前单元格高亮 |
| 单位换算页 | 双栏输入 | 左右对称布局、中间双向箭头、下拉选择单位 |
| 历史与收藏页 | 列表展示 | 卡片式列表、悬浮显示详情、滑动删除 |

### 4.3 响应式设计
- 桌面端优先（1280px+）：完整布局，键盘区 + 侧边栏
- 平板端（768px-1279px）：键盘区折叠为可展开面板
- 移动端（<768px）：全屏键盘模式，顶部切换标签
